A) Quick Start Guide:

Simplest method:

If your Adobe Muse version is greater or equal than 2015.1.2.44 follow 3 easy steps!

 

  1. Unpack the widget zip file with WinRAR or other program for working with .zip files.
  2. Open with Adobe Muse the 3D-Tour-examples.muse file, which located in examples folder.
  3. Follow instructions in this file

Widgets installation:

  1. Unpack the widget zip file with WinRAR or other program for working with .zip files.
  2. Double-click the 3D-Panorama_pack_EN.mulib file to import it into Muse. Alternatively, you can use the Import button in the Library panel within Muse and browse to the file on your hard drive.
  3. Copy all files from 'assets' folder to your project assets folder. If you don't have assets folder in your project - create and rename her to 'assets'. Important : store all project assets near Muse file to create a quality project in Adobe Muse. It will avoid problems with resources when exporting your site!

Using the 3D-panorama widget:

  1. Go to the Library Panel in Muse.
  2. Open the 3D-Panorama_pack_EN widget folder
  3. Click and drag the 3D-Panorama widget onto the design canvas
  4. In widget settings panel attach all needed files from your project assets folder to widget and set other settings to your taste. More about widget setting here.
  5. If you want create a 3D-Tour - please, follow this instructions.
  6. Preview and publish the page and enjoy the beautiful and fast 3D-Panorama!

C) 3D-Tour Creation

This is tutorial for creation 3D-Tour. It is believed that page already have needed 3D-panorama, if not - at the beginning place widget 3D-Panorama on your page.

These templates also available in Adobe Muse 2015.1.2.44 project file, which placed in "Examples" folder or  you can get it here

On this page the word "composition" means a set of widgets working together (widget 3D-Panorama together with the 'Image Button' and / or 'Text Button' widgets)

Binding the Widgets

To bind the button widget to 3D-Panorama widget you need to write the same values in 'ID' fields, which located in bottom of the widget settings panel. Symbols of the value must be only 'A'-'Z' or '_' . This values must be unique for each 3D compositions on your page. Example: need to place two 3D-Tour on page - in first Composition use value "panorama_view", in second - "panorama_view_2"

3D-Panorama widget & Text Button widgets

  1. Place the "Text Button" widget on your page
  2. Set a 3D photo in widget settings
  3. Customize the button style.
  4. Binds the button to the needed 3D-Panorama widget
  5. Repeat this steps for each 3D photo

3D-Panorama widget & Image Button widgets

  1. Place the  "Image Button" widget on your page
  2. Set a 3D photo in widget settings
  3. Set a background image using a standard Adobe Muse tools
  4. Bind the button to the needed 3D-Panorama widget
  5. Repeat this steps for each 3D photo

3D-Panorama widget & Image Button widgets with the house plan

  1. Place the transparent rectangle over the 3D-Panorama widget.
  2. Place a PNG image with house plan over rectangle.
  3. Place the  "Image Button" widget in needed position on the plan.
  4. Set the widget size to 30x30 and corner radius for each angle to 30
  5. Remove/replace default fill image and set a background color using a standard Adobe Muse tools
  6. Bind the button to the needed 3D-Panorama widget
  7. Repeat this steps for each 3D photo

D) 3D-Image Optimizations

3D image restrictions:

 

  • The ratio of width and height must be strictly 2 to 1. If for some reasons you have decided to use the photo with incomplete panoramic view you need to increase the height of the picture to 1 / 2 of its width in any graphics program ( in the panorama should get a white / black region above and below).
  • Image resolution shall not exceed 10,000 x 5,000 pixels ( especially WebGL )
  • An image that is larger than 4 MB is not recommended to use because large photos will significantly reduce the speed of loading your site.
  • Your site mobile version must have 3D photo with size less than 2000 x 1000 or more than 3500 x 1750 pixels. This is due to the fact that the drop-down period permissions correctly displayed on some versions of iPhone. (Linked to the implementation of WebGL support on iPhones)

It is established experimentally that the best balance of image quality and download speed is achieved when the size of

the 3D file photo

 

  • In computer version - from 1 to 2 MB and the resolution of 4000 x 2000 pixels
  • In mobile version - about 500 KB and the resolution of 2000 x 1000 pixels

Other tips:

  • You can read more about creation a 3D-Photo here.
  • Recommended 3D photos creating program is Image Composite Editor (ICE). Also you can make 360 degrees render in Program for work with 3D graphic. At example - Tutorial: 360° VRay for SketchUp Rendering
  • To quickly and easily create thumbnails from 3D photo open your panorama in a browser, set a good camera angle and select right click - "save image as"
