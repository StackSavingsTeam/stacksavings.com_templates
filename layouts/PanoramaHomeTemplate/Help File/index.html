About Template

Home Panorama is a professional Adobe Muse theme for a real estate review. Template designed for personal use and real estate agents. Purchase includes the 3D Panorama widget for Adobe Muse. Widget's page is: https://codecanyon.net/item/3dpanorama-pack-for-adobe.

Template has organized layers panel that makes it easy to edit. Mobile and tablet versions are also included.

Template developed with the Muse CC 2017.0.1.11.

B. How to use fonts

You have to add fonts immediately after opening the Muse file. You can watch the video about add self-hosted web fonts in Adobe Muse on the official page https://helpx.adobe.com/muse/how-to/add-self-hosted-web-fonts.html.

C. Layers Panel

1. Open Layers panel

Open layers panel select the Adobe Muse menu item Window - Layers or press F7 on keyboard.

2. Working with layers

Template has organized layers panel that helps to edit the theme convenient and easy. Layers can be visible, hidden, and blocked. To create or delete a layer click on the corresponding icon at the bottom of the layers panel. To edit any element of the template select a desired layer and look for the element you want to modify.

M. Publish Website

To publish site select the Adobe Muse menu item File - Upload to FTP Host or press Ctrl+Alt+F on keyboard. You will see the window displayed below.

In the fields you have to enter the details of your hosting. These data can be viewed in your account hosting or letter that came in the mail after the purchase account. Then click Next.

Then will appear a window in which you need to enter the URL of your site. After clicking on the OK the site will be published.
